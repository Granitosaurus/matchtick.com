API usage:

matchtick.com provides a lightweight and convenient http json api for esport match and tournament data.

Getting match data:

To retrieve data for matches use /tick/(game_name) end point. Where (game_name) can be anything of this list:

['dota2', 'counterstrike', 'hearthstone', 'heroesofthestorm', 'lol', 'overwatch', 'starcraft2', 'all']

For example to retrieve the latest dota2 matches you can use http://matchtick.com/tick/dota2.

Getting tournament data:

To retrieve data for tournaments or other events use /tournament/(game_name) end point. Where (game_name) can be anything of this list:

['dota2', 'counterstrike', 'overwatch']
