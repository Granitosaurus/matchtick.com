About:

matchtick.com is a lightweight and convenient api for esport match data. The data is supplied by gosugamers.org and liquidpedia.net. The data is updated every 5 minutes from these sources and all credit related to the data goes to these sources.

It's built in python with flask and ggmt and Is licensed under copy-left GPLv3 license.

The source code and everything else is available for free on https://github.com/Granitosaurus/matchtick.com. Pull requests, suggestions, ideas and feedback is very much welcome! The website is hosted on linode.com.
